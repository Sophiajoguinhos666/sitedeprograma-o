As tags do HTML são usadas para informar ao navegador a estrutura do site e demarcar textos. Elas consistem de vários parênteses com seus nomes e abreviações para cada código inserido, e dentre eles À códigos que são essenciais dentro do HTML, que são:

html: que é o mais importante pois ele vai indicar todo o caminho a ser trilhado.

head: é resumidamente o cabeçalho do projeto.

title: titulo da página.

body: corpo/estrutura da página.

h1 a h6: é o tamanho da fonte (seus títulos e subtítulos).

p: define seu parágrafo.

a: define hyperlink (Um recurso que pode ser outro documento da web, uma imagem, um vídeo, um áudio e etc).

img: define uma imagem.

ul e li: uma lista não ordenada ( usada para apresentar informações em uma lista de itens que não tem uma ordem específica)

table: define o início e o fim da tabela.

tr: define uma linha dentro de uma tabela criada com a tag table.

th: cabeçalho da tabela.

td: os dados de uma tabela.

Além do HTML, também temos o CSS, que é uma linguagem de estilo utilizada para descrever a apresentação de um documento descrito e uma linguagem de marcação, como o HTML, XML ou XHTML Exemplo: selecionar elemento por tags, classes, id, seus elementos filhos e selecionar elementos dentro de uma elemento especifico.
